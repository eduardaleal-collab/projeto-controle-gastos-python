# Controle de Gastos em Python

Meu primeiro projeto em Python.

## Objetivo
Criar um programa que registre gastos e faça cálculos automaticamente.

## Funcionalidades
- Receber gastos do usuário
- Calcular total
- Calcular média
- Mostrar maior gasto
- Mostrar menor gasto

## Tecnologias utilizadas
- Python

## Como executar

Execute:

python controle_gastos.py
